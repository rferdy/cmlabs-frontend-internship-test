# cmlabs-frontend-internship-test
CookingMaster merupakan sebuah website yang berisikan resep-resep berbagai masakan dari berbagai negara di dunia.

# Index
berisi category makanan yang ada. klik salah satu category dan anda akan diarahkan ke halaman category

# Category
berisikan meals yang termasuk dalam suatu category. klik salah satu menu dan anda akan diarahkan ke halaman meal.

# Meal
berisikan resep makanan yang anda pilih.
